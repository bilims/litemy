# litemy
Plain logic data converter for SQLite into MySQL in PHP.

## usage

input: http://{HOST}/litemy.php?schema={SCHEMA_SQLITE.SQL}&data={DATA_SQLITE.SQL}

output: *1-import_schema.mysql.sql* and *1-import_data.mysql.sql*
